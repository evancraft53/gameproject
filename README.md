## Game Idea (Full-44)

### Concept
Diddy Bird is a parody-style game inspired by Flappy Bird with an analog horror twist. The game mixes humor with unsettling visuals to create a chaotic and funny experience.

### Gameplay
The game plays almost exactly like Flappy Bird. The player taps to keep the bird in the air while navigating through obstacles.

### Features
- Simple one-button controls
- Endless gameplay
- Increasing difficulty over time
- Parody + analog horror aesthetic

### Unique Twist
Instead of normal pipes, the obstacles are surreal/parody-style visuals (like strange or exaggerated characters), creating a mix of humor and absurdity.
